CS Classroom



Our aim is to build a platform that will help in field of computer science education. The project will be comprised of various educational toolkits such as educational blog, course management web service. Our main concern is computer science education. So there will be modules specific to this field like dataset collector, online judge, code generator.

Tutorial: This module will allow people to share their articles, tutorial series. Students will be able to access the articles on the web. We plan for making an extension which will automatically generate questions based on a article. It will help students to judge their understandings of the article. Other extensions like recommendation system, search, automatic hashtag generation will also be there.

Course Manager: This module will help teachers to manage their courses and communicate with students easily.

Dataset Collector: Collecting data is an important in computer science especially in the field of Human Computer Interaction. The aim is to make this process more suitable.

Algorithm Visualization: Learning Algorithms from pseudocode is hard. Simulating is always more helpful in classrooms. We will build simulation tools for that.

Code Helper: This module will automatically generate codes for various machine learning libraries. Besides, there will be a judge for checking implementations of popular algorithms.
