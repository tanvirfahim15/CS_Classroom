Given a limited supply of coins of given denominations, you have to determine whether it is possible to make change for a particular amount or not.

You can use a coin at most once.

For example,

C = { 1 , 2 , 3 }, N = 4

Yes. It is possible to make change for amount of 4 taka


C = { 1 , 2 , 5 }, N = 4

No. It is not possible to make change for amount of 4 taka

{ 1, 3 }


C = { 1 , 2 , 3, 4 }, N = 5

C = { 1 , 3, 5 , 7}
